# Raspberry Pi HD Camera v1.0
This is an HD camera for raspberry pi

## V4L2 Parameters Available

### Pixel Formats
  YU12 - Planar YUV 4:2:0
  YUYV - YUYV 4:2:2
  RGB3 - 24-bit RGB 8-8-8
  JPEG - JFIF JPEG
  H264 - H.264
  MJPG - Motion-JPEG
  YVYU - YVYU 4:2:2
  VYUY - VYUY 4:2:2
  UYVY - UYVY 4:2:2
  NV12 - Y/CbCr 4:2:0
  BGR3 - 24-bit BGR 8-8-8
  YV12 - Planar YVU 4:2:0
  NV21 - Y/CrCb 4:2:0
